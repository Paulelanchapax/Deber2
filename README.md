# Deber2
Conversor de Divisas (Deber 1)
Este deber consistio en el desarrollo de un programa en C++ diseñado para realizar conversiones entre diferentes divisas. El objetivo principal es aplicar la lógica de programación y estructuras de control para gestionar entradas del usuario, realizar cálculos matemáticos y validar datos de manera eficiente.

Lógica del codigo:

Entrada: Solicita divisas (valida 3 letras), tasa de cambio y monto (valida valores positivos).

Proceso: Calcula el monto convertido y solicita un timestamp.

Salida: Muestra el reporte Market Snapshot con los detalles de la operación.

Control: El usuario decide si repetir el proceso o finalizar el programa.

Incluye código fuente y diagrama de flujo.
